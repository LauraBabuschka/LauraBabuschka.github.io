More than just a font, Badsans invites designers to explore and play with the harmony between nature and design. Badsans is a display typeface inspired by the organic flow of melting snow and ice, featuring irregular edges and an interesting texture.

BADSANS CAN BE USED TO CREATE AN INTERESTING VISUAL FOR QUOTES AND LOOKS GOOD IN ALL CAPS.
Although this font was designed to be used as display font it is still leggible when used for continuous text. When used for running text it creates a very interesting texture.
